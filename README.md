# mxshop

从网上找了一些视频解析借口，写了个简单的网页，除了支持视频解析之外，还支持时间显示，天气预报等功能，位置信息保存在 localstorage 中，自动读取，不用每次打开网页都设置

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

```
yarn run build
```
